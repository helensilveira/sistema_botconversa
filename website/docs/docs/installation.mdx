# Installation

Install the component via [npm](https://www.npmjs.com/):

```bash
npm install botconversa-chat
```

For React, install the following instead: <br />

```bash
npm install botconversa-chat-react
```

Access the component via CDN:

```
https://unpkg.com/botconversa-chat@1.4.11/dist/botconversaChat.bundle.js
```
